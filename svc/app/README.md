
## 目录结构
```
├── api                 路由
├── client              http客户端
├── core                设置、异常等系统基础项
├── crud                crud抽象
├── db                  数据库连接、配置
├── gunicorn_conf.py    gunicorn配置
├── __init__.py         __init__.py
├── logger              日志配置
├── main.py             程序入口
├── middleware          中间件
├── models              数据库模型
├── scheduler           定时任务
├── schemas             业务模型
├── tests               测试
└── util                工具
```
